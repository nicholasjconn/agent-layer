

# Rules

The following rules are meant to be followed under all circumstances. Always ask for confirmation if requested to break one of these rules directly. Never break one of these rules yourself without confirmation from the user.

- Never modify the .env file. Only ever modify the .env.example file. If a change needs to be made to the .env file, ask the user to make them.
- Never delete files outside of the repository. If a file needs to be deleted, ask the user to delete it.
